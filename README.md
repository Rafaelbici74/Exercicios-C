# 💻 Estudos e Práticas em Linguagem C

Este repositório contém uma coleção de exercícios e programas desenvolvidos durante o estudo da linguagem **C**, baseados na leitura e prática do livro **"Linguagem C", 10ª edição, de Luís Damas**.

O projeto reúne as atividades práticas realizadas para consolidar os conceitos de programação estruturada, partindo da lógica inicial até a implementação de algoritmos mais complexos.

---

# 📚 Informações Acadêmicas

**Referência:** Linguagem C (Luís Damas, 10ª Edição)  
**Curso:** Desenvolvimento de Sistemas  
**Objetivo:** Transição de lógica de programação (Portugol) para a sintaxe da linguagem C  

---

# 🛠️ Tecnologias Utilizadas

Este projeto foi desenvolvido utilizando as seguintes tecnologias e ferramentas:

### Linguagem C
Linguagem de programação de base, utilizada para compreender o funcionamento de baixo nível, alocação de memória e performance.

### Compilador (GCC)
Utilizado para a compilação e geração dos binários a partir do código-fonte.

### Lógica de Programação
Aplicação de conceitos estruturados como tomada de decisão, laços de repetição e modularização através de funções.

---

# 🧩 Estrutura do Projeto

A organização dos arquivos segue o progresso do aprendizado prático proposto pela literatura base.

## src

A pasta `src` contém o código principal das atividades.

### exemplos

Contém os exemplos de código apresentados durante a leitura para demonstração de sintaxe.

Estes arquivos cobrem:
- Entrada e saída de dados (`stdio.h`)
- Declaração de tipos de variáveis
- Utilização de operadores aritméticos e lógicos

### atividades

Contém as resoluções dos exercícios propostos ao final de cada capítulo do livro.

As atividades envolvem:
- Estruturas de controle (`if/else`, `switch`)
- Laços de repetição (`for`, `while`, `do-while`)
- Manipulação de vetores, matrizes e strings
- Uso de ponteiros e estruturas de dados

# ▶️ Como Executar o Projeto

Para compilar e testar os códigos, é necessário ter um compilador C instalado (como o **GCC**).

Passos para execução:

1. Clonar o repositório para sua máquina local.
2. Abrir o terminal na pasta do arquivo desejado.
3. Compilar o código:
   ```bash
   gcc nome_do_arquivo.c -o programa

Executar o aplicativo:

No Windows: programa.exe

No Linux/Mac: ./programa
